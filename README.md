# ♟️ Chess System (Java)

O sistema simula um jogo de xadrez completo no console, implementando regras reais do jogo, como movimentação de peças, validações, xeque e xeque-mate.

---

## 🚀 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Estrutura de projetos em camadas

---

## 📁 Estrutura do projeto

```
src
 ├── application
 │    ├── Program.java
 │    └── UI.java
 │
 ├── boardgame
 │    ├── Board.java
 │    ├── BoardException.java
 │    ├── Piece.java
 │    └── Position.java
 │
 └── chess
      ├── ChessException.java
      ├── ChessMatch.java
      ├── ChessPiece.java
      ├── ChessPosition.java
      ├── Color.java
      └── pieces
```

---

## 🎯 Funcionalidades

* Movimentação de peças conforme regras do xadrez
* Validação de jogadas
* Identificação de xeque
* Identificação de xeque-mate
* Controle de turnos entre jogadores
* Tratamento de exceções personalizadas

Durante o desenvolvimento, alguns dos principais desafios foram:

* Modelar corretamente as regras do xadrez
